# Backend – Next.js + Supabase

Backend REST desenvolvido com Next.js e Supabase para gerenciamento de usuários, serviços, categorias, cidades e upload de imagens.

## Tecnologias
- Next.js
- TypeScript
- Supabase (PostgreSQL, Auth, Storage)

## Funcionalidades
- CRUD de usuários
- CRUD de serviços
- CRUD de categorias
- CRUD de cidades
- Upload de avatar e logo

## Configuração

Crie o arquivo `.env.local`:

NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
SUPABASE_SERVICE_ROLE_KEY=

## Execução

npm install  
npm run dev  

API disponível em http://localhost:3000
